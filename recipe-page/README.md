## 📄 Project Overview

This project is part of the **Responsive Web Design Certification** on freeCodeCamp.org. It focuses on practicing the fundamentals of HTML by creating a structured recipe webpage.

The goal of this project was to build a simple recipe page that displays a meal introduction, ingredients list, cooking instructions, and an image while following proper HTML structure and accessibility practices.

## ✅ What I’ve Learned

By completing this project, I have learned how to:

- Create a complete HTML5 document structure
- Use headings to organize webpage content (`h1`-`h2`)
- Add text content using paragraph elements
- Create ingredient lists using unordered lists (`ul`)
- Create step-by-step instructions using ordered lists (`ol`)
- Add images using the `img` element
- Use `src` attributes to load images
- Use `alt` attributes for image accessibility
- Structure webpages with clean and readable HTML
