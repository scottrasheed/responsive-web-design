## 📄 Project Overview

This project is part of the **Responsive Web Design Certification** on freeCodeCamp.org. It focuses on building a travel agency webpage using semantic HTML elements and proper document structure.

The goal of this project was to create a travel destination page that introduces Japan, displays travel packages, and showcases popular itineraries using images, links, and structured content.

## ✅ What I’ve Learned

By completing this project, I have learned how to:

- Create a complete HTML5 document structure
- Use semantic HTML elements to organize webpage content
- Add SEO information using the `meta` description tag
- Create sections using heading elements (`h1`-`h2`)
- Create navigation links using anchor elements (`a`)
- Use `href` and `target="_blank"` attributes
- Organize content using unordered lists (`ul`)
- Display images using the `img` element
- Create image sections using `figure` and `figcaption`
- Add meaningful `alt` text for accessibility
